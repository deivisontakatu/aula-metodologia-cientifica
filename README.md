# 📘 Metodologia Científica — Guia para TCC/TG

Bem-vindo(a) ao repositório de apoio à disciplina de Metodologia Científica.

Este material foi desenvolvido para orientar, de forma prática e estruturada, o planejamento e desenvolvimento de trabalhos acadêmicos, especialmente TCC (Trabalho de Conclusão de Curso) e TG (Trabalho de Graduação).

---

## 🎯 Objetivo do repositório

Este repositório tem como objetivo:

- Servir como **guia estruturado** para elaboração do TCC/TG
- Apoiar o desenvolvimento de cada etapa da pesquisa científica
- Fornecer **modelos, exemplos e boas práticas**
- Garantir **coerência metodológica e rigor acadêmico**

---

## 🧱 Estrutura do trabalho

O desenvolvimento do TCC/TG deve seguir a estrutura organizada nos arquivos deste repositório:

### 📄 01 - Tema, Problema e Objetivos
- Definição do tema
- Formulação do problema de pesquisa
- Objetivos geral e específicos
- Justificativa

---

### 📄 02 - Revisão Bibliográfica
- Fundamentação teórica
- Conceitos principais
- Trabalhos relacionados
- Identificação de lacunas

---

### 📄 03 - Metodologia
- Tipo de pesquisa
- Delineamento
- Coleta de dados
- Análise dos dados
- Aspectos éticos

---

### 📄 04 - Resultados e Discussão
- Apresentação dos dados
- Análise e interpretação
- Comparação com a literatura

---

### 📄 05 - Conclusão e Considerações Finais
- Síntese dos resultados
- Resposta ao problema
- Contribuições
- Trabalhos futuros

---

## 🧭 Como utilizar este repositório

Para um melhor aproveitamento:

1. **Siga a ordem dos arquivos (01 → 05)**
2. Utilize cada arquivo como **template do seu trabalho**
3. Preencha progressivamente cada seção
4. Revise constantemente para manter coerência entre as partes

⚠️ Importante:  
O TCC não deve ser feito de forma isolada por partes. Todas as seções devem estar conectadas:

- O problema define os objetivos  
- Os objetivos orientam a metodologia  
- A metodologia gera os resultados  
- Os resultados sustentam a conclusão  

---

## 🧠 O que você vai desenvolver

Ao utilizar este material, espera-se que você seja capaz de:

- Formular problemas de pesquisa claros e relevantes  
- Construir uma revisão bibliográfica crítica  
- Definir métodos adequados de pesquisa  
- Coletar e analisar dados com rigor  
- Redigir textos científicos com clareza  

---

## 🛠️ Recursos recomendados

Durante o desenvolvimento do trabalho, recomenda-se o uso de:

- **Gerenciadores de referência:** Zotero, Mendeley  
- **Ferramentas de escrita:** Word, Google Docs, LaTeX  
- **Análise de dados:** Excel, R, Python  
- **Organização:** Notion, Obsidian  

---

## ✅ Checklist de desenvolvimento

Antes de avançar, verifique:

- [ ] O problema está claro e bem delimitado?  
- [ ] Os objetivos estão coerentes com o problema?  
- [ ] A revisão bibliográfica está atualizada?  
- [ ] A metodologia está bem definida?  
- [ ] Os resultados respondem ao problema?  
- [ ] A conclusão está alinhada com os objetivos?  

---

## ⚖️ Boas práticas acadêmicas

- Evite plágio (sempre cite fontes)
- Utilize normas da instituição (ABNT ou APA)
- Mantenha clareza e objetividade na escrita
- Revise o texto antes da entrega
- Garanta consistência entre as seções
